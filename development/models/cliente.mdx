## Overview

The `Cliente` model represents clients in the VIP2CARS system. It handles client information including personal details, contact information, and manages the relationship with their vehicles.

## Relationships

### vehiculos()

Defines a one-to-many relationship with the Vehiculo model. A client can have multiple vehicles.

**Relationship Type:** `hasMany`

**Related Model:** `App\Models\Vehiculo`

**Foreign Key:** `id_cliente`

**Local Key:** `id_cliente`

```php
public function vehiculos()
{
    return $this->hasMany(Vehiculo::class, 'id_cliente', 'id_cliente');
}
```

## Usage Examples

<CodeGroup>

```php Create a Client
use App\Models\Cliente;

$cliente = Cliente::create([
    'nombres' => 'Juan Carlos',
    'apellidos' => 'Pérez García',
    'nro_documento' => '12345678',
    'correo' => 'juan.perez@example.com',
    'telefono' => '+51987654321',
]);
```

```php Get Client with Vehicles
use App\Models\Cliente;

$cliente = Cliente::with('vehiculos')->find($id);

// Access client's vehicles
foreach ($cliente->vehiculos as $vehiculo) {
    echo $vehiculo->placa;
}
```

```php Update Client
use App\Models\Cliente;

$cliente = Cliente::findOrFail($id);
$cliente->update([
    'telefono' => '+51999888777',
    'correo' => 'new.email@example.com',
]);
```

```php Query Clients
use App\Models\Cliente;

// Get all clients
$clientes = Cliente::all();

// Find by document number
$cliente = Cliente::where('nro_documento', '12345678')->first();

// Get clients with vehicle count
$clientes = Cliente::withCount('vehiculos')->get();
```

</CodeGroup>
