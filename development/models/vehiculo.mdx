## Overview

The `Vehiculo` model represents vehicles in the VIP2CARS system. It stores vehicle details such as license plate, make, model, and year, and maintains the relationship with the vehicle's owner (Cliente).

<ParamField path="id_cliente" type="integer" required>

## Relationships

### cliente()

Defines a many-to-one relationship with the Cliente model. Each vehicle belongs to one client.

**Relationship Type:** `belongsTo`

**Related Model:** `App\Models\Cliente`

**Foreign Key:** `id_cliente`

**Owner Key:** `id_cliente`

```php
public function cliente()
{
    return $this->belongsTo(Cliente::class, 'id_cliente', 'id_cliente');
}
```

## Usage Examples

<CodeGroup>

```php Create a Vehicle
use App\Models\Vehiculo;

$vehiculo = Vehiculo::create([
    'placa' => 'ABC-123',
    'marca' => 'Toyota',
    'modelo' => 'Corolla',
    'anio_fabricacion' => 2023,
    'id_cliente' => 1,
]);
```

```php Get Vehicle with Owner
use App\Models\Vehiculo;

$vehiculo = Vehiculo::with('cliente')->find($id);

// Access vehicle owner information
echo $vehiculo->cliente->nombres;
echo $vehiculo->cliente->correo;
```

```php Update Vehicle
use App\Models\Vehiculo;

$vehiculo = Vehiculo::findOrFail($id);
$vehiculo->update([
    'placa' => 'XYZ-789',
    'anio_fabricacion' => 2024,
]);
```

```php Query Vehicles
use App\Models\Vehiculo;

// Get all vehicles with owners
$vehiculos = Vehiculo::with('cliente')->get();

// Find by license plate
$vehiculo = Vehiculo::where('placa', 'ABC-123')->first();

// Get vehicles by brand
$toyotas = Vehiculo::where('marca', 'Toyota')->get();

// Get vehicles manufactured after 2020
$recent = Vehiculo::where('anio_fabricacion', '>=', 2020)->get();
```

</CodeGroup>
